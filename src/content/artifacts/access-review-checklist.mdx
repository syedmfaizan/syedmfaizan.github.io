## Before starting

- Identify systems in scope.
- Define access grant types.
- Confirm ownership metadata.
- Define usage signals.
- Decide notification policy.
- Decide revocation policy.

## During rollout

- Start in report-only mode.
- Measure false positives.
- Notify owners before enforcement.
- Track user friction.
- Keep audit logs.
